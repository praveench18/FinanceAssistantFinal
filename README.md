# 💰 Finance Assistant

A full-stack personal finance management web application that helps users manage expenses, set budgets, track savings, generate reports, and securely manage their accounts.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- Secure Login
- Forgot Password using OTP
- Reset Password
- Password Hashing (bcrypt)

### 💳 Expense Management
- Add Expense
- Edit Expense
- Delete Expense
- Search Expenses
- Filter by Category
- Sort Expenses

### 📊 Dashboard
- Expense Overview
- Monthly Expense Chart

### 💰 Budget Management
- Set Monthly Budget
- Track Remaining Budget

### 🎯 Savings
- Set Savings Goal
- Track Savings Progress

### 📄 Reports
- Monthly Expense Report
- Category Summary
- Export as PDF
- Export as Excel

---

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder.

Example:

```
EMAIL_USER=your_email
EMAIL_PASS=your_app_password
PORT=5000
```
